# React Playground

This project was bootstrapped with create-react-app ... 

## Playing with:

- Lifecycle Management
- Component internals
- Geolocation
- JSON fetch from API
- Form elements
- Events
- Styled Components
- State Management

### Get stuff
```
git clone https://github.com/cbroberg/react-playground.git
```

### npm install stuff
```
npm install
or
sudo npm install 
```

### yarn install stuff
```
yarn install
or
sudo yarn install 
```

### run stuff
```
npm start
or
yarnpkg start 
```
